DEPLOY.PROTOCOL // 展開手順

Instalação

Operacional em menos de 5 minutos.

Sistema Operacional — Riot API Integrada システム稼動中
01

Clone o repositório

Ou baixe o VisionLOL.exe direto da release mais recente.

02

Instale as dependências

Python 3.10+ necessário. Execute pip install -r requirements.txt.

03

Configure a Riot API Key

Obtenha sua chave em developer.riotgames.com e insira no config.json.

04

Execute

Rode python gui_app_integrated.py e configure o roster na aba Time.

# // CLONE
$ git clone https://github.com/Gor0d/VisionLOL.git
$ cd VisionLOL

# // AMBIENTE VIRTUAL
$ python -m venv .venv
$ .venv\Scripts\activate

# // DEPENDÊNCIAS
$ pip install -r requirements.txt

# // CONFIGURAÇÃO
$ cp config.example.json config.json
# edite config.json com sua Riot API key

# // INICIAR
$ python gui_app_integrated.py
