Funcionalidades

Tudo que um coaching staff precisa em uma só ferramenta

Roster do Time

Gerencie 5 jogadores com roles, múltiplas contas e indicador ao vivo de quem está em partida.

Radar de Performance

6 métricas normalizadas por role — KDA, CS/min, Visão, Gold, Damage Share e Kill Participation.

Dashboard de Partidas

Histórico detalhado, pool de campeões com WR% e mapa de calor gerado a partir das timelines.

Replay Viewer

Visualização quadro a quadro das posições dos jogadores, objetivos, wards e movimentação por lane.

Scrims Dashboard

Registro e comparativo de sessões de scrim contra adversários com métricas por partida.

Customização do Time

Banner com nome, legenda, cor e logo personalizada. Persiste automaticamente entre sessões.

Instalação

Pronto para uso em menos de 5 minutos

1

Clone o repositório

Ou baixe o VisionLOL.exe direto da release mais recente.

2

Instale as dependências

Python 3.10+ necessário. Um pip install -r requirements.txt resolve tudo.

3

Configure a Riot API Key

Copie config.example.json para config.json e insira sua chave do developer.riotgames.com.

4

Execute

Rode python gui_app_integrated.py e configure o roster na aba Time.

# Clone
git clone https://github.com/Gor0d/VisionLOL.git
cd VisionLOL

# Ambiente virtual
python -m venv .venv
.venv\Scripts\activate

# Dependências
pip install -r requirements.txt

# Configuração
cp config.example.json config.json
# edite config.json com sua Riot API key

# Iniciar
python gui_app_integrated.py
